Scientific Software

mssa_matlab

A MATLAB toolbox implementing Multichannel Singular Spectrum Analysis (MSSA) for the analysis of multivariate climate and geophysical datasets. The toolbox includes routines for

  • Principal Component Analysis (PCA) preprocessing
  • Multichannel Singular Spectrum Analysis (MSSA)
  • Monte Carlo significance testing
  • Mode reconstruction
  • Reconstruction onto the original spatial grid
  • Phase composite analysis

Availability: Currently available upon request. If you are interested in using the toolbox for research or teaching, please contact me by email.

mssa_python

mssa_python is a Python implementation of Multichannel Singular Spectrum Analysis (MSSA), developed as a modern alternative to the MATLAB toolbox. The package is designed for efficient analysis of multivariate geophysical and climate datasets and integrates seamlessly with the Python scientific ecosystem.

The toolbox supports reproducible scientific workflows using NumPy, SciPy, xarray, and Matplotlib, making it suitable for large NetCDF datasets.

Features

  • Principal Component Analysis (PCA) preprocessing
  • Multichannel Singular Spectrum Analysis (MSSA)
  • Monte Carlo significance testing
  • Mode reconstruction
  • Spatial reconstruction of climate fields
  • Phase composite analysis
  • Native support for NumPy and xarray

Availability: Currently under active development. Please contact me if you are interested in testing the package or contributing to its development.
