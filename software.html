ConvMLR Suite

ConvMLR Suite is a Python framework for hybrid physics–machine learning prediction of weather and climate. Originally developed for subseasonal-to-seasonal prediction of the Indian Summer Monsoon, the framework combines convolutional feature extraction with multi-linear regression to learn spatio-temporally varying weights for multi-model ensemble forecasts.

Unlike conventional ensemble averaging, ConvMLR exploits the strengths of individual dynamical models while correcting systematic regional and temporal biases. The framework provides an interpretable and computationally efficient approach for statistical post-processing and weighted ensemble prediction of geophysical variables.

Features

  • Hybrid physics–AI prediction framework
  • Weighted multi-model ensemble learning
  • Convolutional spatial feature extraction
  • Spatio-temporal multi-linear regression
  • Cross-validation and model evaluation
  • Forecast verification and skill assessment
  • Support for gridded climate and weather datasets
  • Built using PyTorch, NumPy, xarray, and SciPy

Applications: Subseasonal-to-seasonal prediction, monsoon forecasting, climate variability, Earth system model post-processing, and weighted ensemble prediction.

Availability: Currently under active development. Please contact me if you are interested in testing the package or contributing to its development.

mssa_python

mssa_python is a Python implementation of Multichannel Singular Spectrum Analysis (MSSA), developed as a modern alternative to the MATLAB toolbox. The package is designed for efficient analysis of multivariate geophysical and climate datasets and integrates seamlessly with the Python scientific ecosystem.

The toolbox supports reproducible scientific workflows using NumPy, SciPy, xarray, and Matplotlib, making it suitable for large NetCDF datasets.

Features

  • Principal Component Analysis (PCA) preprocessing
  • Multichannel Singular Spectrum Analysis (MSSA)
  • Monte Carlo significance testing
  • Mode reconstruction
  • Spatial reconstruction of climate fields
  • Phase composite analysis
  • Native support for NumPy and xarray

Availability: Currently under active development. Please contact me if you are interested in testing the package or contributing to its development.

mssa_matlab

A MATLAB toolbox implementing Multichannel Singular Spectrum Analysis (MSSA) for the analysis of multivariate climate and geophysical datasets. The toolbox includes routines for

  • Principal Component Analysis (PCA) preprocessing
  • Multichannel Singular Spectrum Analysis (MSSA)
  • Monte Carlo significance testing
  • Mode reconstruction
  • Reconstruction onto the original spatial grid
  • Phase composite analysis

Availability: Currently available upon request. If you are interested in using the toolbox for research or teaching, please contact me by email.
